密室逃脫

密閉空間
逃出房間

密室逃脫一般泛指一種特定的遊戲類型，玩家的視角通常以第一視角為主（或以採用笑第三視角扮演主人公的模式），並且往往被限定在一個近乎完全封閉或者對自身存在威脅的環境內，需要發現和利用身邊的物品（工具），完成指定任務（通常是解開特定謎題的方式），最終達到逃離該區域的目的。

玩密室逃脫的優點？

觀察力

邏輯思考

直覺聯想

透過遊戲過程訓練邏輯思考、團隊合作以及找到自我定位，脫逃遊戲，是一個需要運用邏輯思考及團隊合作的一種實境益智遊戲，其中不只具備娛樂性，更能訓練邏輯思考與合作互助，在輕鬆的遊戲過程中提升自身能力、凝聚團隊向心力。透過遊戲，大家更容易展現真實的反應，可以讓玩家認識到自己的優點與缺點，幫助玩家在遊戲過程中找到自己在團隊中的定位、增加團隊情感與向心力。

企業培訓

在職進修與同樂會等場合中，用活潑的學習體驗構成深刻的印象，與平日的工作形成強力的連結
→ 於限定時間內與其他成員一起往共同目標邁進，體驗團隊合作的意義。
→ 與團隊成員間共享資訊，互通有無。
→ 扮演好各自的角色，相互彌補不足的地方，充分養成團隊合作的精神。
→ 藉由溝通的方式，了解團隊成員的大小事。
→ 可以體會到達成目標是一件非常不容易的事。
→ 與傳統八股的在職進修或同樂會不同，用獨特有趣的方式建立起團隊合作。
